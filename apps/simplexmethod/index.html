Description

Thanks for checking out my Row Reduction app. Check out Simplex Method at the app store now!

Check out a Video Demostration on how to use this app.

Features:

- Solves Standard Max, Standard Min, and Non-Standard

- Let's you work through the method yourself

- Offers hints when requested

- Undo operations if you make a mistake

Need to solve a linear programming problem? Finished using the Simplex Method but you're not getting the right answer? One of the major drawbacks to the Simplex Method is you have to keep track of a page full of tedious calculations AND choose the correct row operations. When you first learn this technique it's hard to do both correctly.

The Simplex Method involves using a sequence of "If/Thens" that lead you to the initial Simplex Table. This app walks you through the set up and creates the initial table. From there you can either use the method on your own or have it help you through Hints. Choose the operation and the app will generate the new table for you. You get to focus solely on practicing and experimenting.

What if you do all this handwritten work and still can't get the answer?

You can also purchase the Instant Solution upgrade. Type in the system of equations and click SOLUTION. It will generate all the correct row operations and tables and display them all at once. Now you can follow along and try to find your mistake.

What I learned...

My second app!

The simplex method app was a follow up to the Row Reduction app. The structure and flow was identical so I was able to reuse a lot of my row reduction code. I wrote that app in such a way so I could reuse most of the code. That's what I learned... writing code in such a way that it can be used in other projects.

What was hard

The difference between the Row Reduction app and the Simplex Method app was writing the code for the hints. The Simplex Method is hard enough to explain and even more challenging to code. I learned that writing out a thorough flowchart for the method helped me create the classes, methods, etc. Basically I learned that you should really plan ahead.

What I could do better

Now that I've learned more I would love to go back and redo this app. I want to clean up the classes dealing with the simplex algorithm. The instant solution option prints out each step of the algorithm. I had to create subviews programatically. And I'm sure I can make it cleaner and more efficient.
